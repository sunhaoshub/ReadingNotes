左值：有地址的值

右值：不能取地址

只能通过右值初始化右值引用，不能通过左值初始化右值引用

右值引用：&&

右值引用只能通过右值初始化

常量左值引用是一个万能的初始化类型

```C++
#include<iostream>
#pragma warning(disable:4996)


int main()
{
	int a = 5;
	int& b = a;
	//int&& c = b; 报错
	int&& c = 6;//不报错
	 return 0;
}
```

```C++
#include<iostream>
#include<string.h>
#include<sstream>
#pragma warning(disable:4996)


int main()
{
	//左值
	int num = 9;
	int num1 = 11;
	//左值引用
	int& a = num;
	//右值引用
	int&& b = 10;
	//常量左值引用
	const int& c = 9;
	const int& d = num;
	const int& e = d;

	//常量右值引用
	const int&& d = 10;
	const int&& f = b;//error
	const int&& g = a;//errror
	const int&& h = c;//error
}
```



右值引用的作用：**延长某块内存的作用时间**；

移动构造的目的就是实现浅拷贝，但是析构的时候不释放内存，令原指针指向null即可，即**内存转移**；

编译器会判断右边的对象是不是临时对象
