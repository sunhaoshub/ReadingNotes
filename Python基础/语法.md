继承：

```python
class DuolaAMeng:
    def __init__(self,name):
        self.name = name
    def getZhuQingTing(self):
        print("给"+self.name+"⼀个⽵蜻蜓")
class DuolaBMeng (DuolaAMeng):
    pass
daxiong = DuolaBMeng("大雄")
daxiong.getZhuQingTing()
```
